# Payment Service

Handles payment request validation and payment status updates.

### 💳 Features

- Verifies payment for each order
- Publishes payment success/failure events via Kafka
- Works closely with order-service

### ⚙️ Tech Stack

- Java, Spring Boot
- Kafka, REST
